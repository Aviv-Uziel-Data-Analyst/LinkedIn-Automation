##  סקירת הפרויקט 🎯

המדריך מדגים כיצד לבנות פייפליין אוטומטי ליצירת קשר בלינקדאין באמצעות **Clay.com** לייצור לידים חכם ו-**LaGrowthMachine (LGM)** להרצת קמפיינים אוטומטיים.
המערכת פותחה כחלק מאסטרטגיית חיפוש עבודה מתקדמת, המשלבת מומחיות בניתוח נתונים עם פלטפורמות אוטומציה שיווקיות.



###  מבנה ה-Pipeline 🔨
בסדר הבא: **[Clay.com](https://www.clay.com/) (ייצור לידים) ← [LaGrowthMachine (LGM)](https://lagrowthmachine.com/) (אוטומציה) ← LinkedIn (המשך שיחה אנושי)**

 תיקיות הפרויקט בנויות בשלבים, מ1 עד 5 ומכילות בתוכן צילומי מסך למי שיותר קל לו לעקוב בצורה ויזואלית.

## 2️⃣ הגדרת מפתח API ב- LGM

הגדר את חיבור ה-API בין LGM לפלטפורמות חיצוניות. שלב זה חיוני לאפשר זרימת נתונים בין כלי ייצור הלידים לפלטפורמת האוטומציה.

#### גישה להגדרות API
לחצו על תמונת הפרופיל שלגם למטה בצד ימין. לאחר מכן לחצו על *Inegrations & API*, העתיקו את הקוד שנמצא שם, זה יאפשר אינטגרציה עם Clay.
שמרו את המפתח הזה במקום נוח, אתם תצטרכו אותו תכף.


**🔒 הערת אבטחה:** לעולם אל תשתף את מפתח ה-API שלך ושמור אותו במקום בטוח.

## 3️⃣ פלטפורמת Clay - הגדרה ראשונית וקונפיגורציה

פלטפורמת **Clay** תשמש כשכבת המודיעין, ימצא ויכשיר לידים לפני שליחתם לפלטפורמת האוטומציה.

#### יצירת workbook חדש ב-Clay
התחילו ביצירת **Workbook** חדש ב-Clay, לייצור הלידים.
לאחר מכן בחרו ב- **Find people** מבין האפשרויות בצד שמאל.
זה מה שמגדיר לפלטפורמה איזה סוג נתונים לייבא לטבלה שלכם.

#### הגדרת מקורות נתונים
כאן תוכלו לבחור כל מיני פילטרים שיעבדו **לפני** שהנתונים נכנסים לטבלה שלכם.
#### -> ⚠ חשוב מאוד⚠

בתווית של *Location*, בתוך **Countries to include** תרשמו **Israel**

אחרת הפלטפורמה תייבא אנשים מחו"ל (אלא אם כן זה המטרה שלכם)

#### סוג משרה
החלק הקריטי הנוסף הוא הפילטר בשם **Job Title**.

בחרו מה שרלוונטי עבורכם - ושימו לב - לכל **TItle** יש המון וריאציות אז תשקיעו זמן בלמצוא את כל השמות הנרדפים למשרה שאתם מחפשים.
במקרה שלי השתמשתי בתפקידי דאטה, ומגייסים (Recruiters, Talent Aquisition, etc)


#### פילטרים נוספים
תוכלו להגדיר כל פילטר אחר שתרצו - לפי תעשייה, סניוריטי ועוד המון דברים - שוב לפי המטרה שלכם.

כשסיימתם להגדיר הכל לחצו על **Continue** ואז **Import to new table**

#### הוספת פיצ'רים לטבלה (עמודות נוספות)
העמודה הראשונה שנוסיף היא id מיוחד לכל ליד.

- נלחץ על **New Column**  ולאחר מכן **Formula**
- נדביק את הנוסחא הבאה: **Math.random().toString(36).slice(2)+Date.now().toString(36)**




#### סינון מתקדם ואימות
הגדר מסננים מתקדמים כדי להבטיח איכות הלידים. הגדר כללי אימות עבור כתובות אימייל, מידע על חברות ורלוונטיות איש הקשר.




#### העשרת נתוני חברות
הוסף העשרת נתונים ספציפית לחברות כדי לספק הקשר לפרסונליזציה. כלול גודל חברה, תעשייה, חדשות אחרונות ומדדי צמיחה.





#### עיצוב פלט והגדרת ייצוא
הגדר את פורמט פלט הנתונים הסופי כדי להבטיח תאימות עם LaGrowthMachine. הגדר טריגרים לייצוא אוטומטי ואימות נתונים.

## 4️⃣ Clay - הגדרה מתקדמת - זרימות עבודה לחיפוש ויצירת לידים

הגדר זרימות עבודה מתקדמות ב-Clay לעיבוד לידים חכם. שלב זה כולל הגדרת לוגיקה מותנית לכישור לידים ופעולות אוטומטיות בהתבסס על איכות הנתונים.

### 🔍 זרימת עבודה לחיפוש לידים

#### הגדרת חיפוש לידים
הגדר את זרימת העבודה לחיפוש לידים כדי לזהות ולאמת פרוספקטים פוטנציאליים. זרימת עבודה זו משתמשת בלוגיקה מותנית כדי לקבוע אם יש לעבד ליד נוסף.

**תנאי הרצת חיפוש ליד:**
{{first_name_hebrew}}?.translation?.translatedText && {{LinkedIn Profile}}

פרמטרי אימות לידים

הגדר כללי אימות כדי להבטיח איכות הלידים. הגדר בדיקות עבור שלמות פרופיל, ניקוד רלוונטיות ודיוק מידע יצירת קשר.
מסנני חיפוש מתקדמים

יישם לוגיקת סינון מתקדמת כדי לחדד את איכות הלידים ולהבטיח שרק פרוספקטים בעלי פוטנציאל גבוה ימשיכו לשלב היצירה.
🏗️ זרימת עבודה ליצירת לידים
לוגיקת יצירת לידים

הגדר את זרימת העבודה ליצירת לידים המעבדת פרוספקטים מכושרים ומכינה אותם לייצוא ל-LaGrowthMachine. זרימת עבודה זו כוללת עיצוב נתונים ואימות סופי.

תנאי הרצת יצירת ליד:

javascript
!{{Search Lead}} && {{Company Name}} && {{LinkedIn Profile}}

עיצוב נתונים וסטנדרטיזציה

הגדר כללי עיצוב נתונים כדי להבטיח עקביות בין כל הלידים. סטנדרטיזה מוסכמות שמות, פורמטי יצירת קשר ומידע על חברות.
הכנה סופית של לידים

השלם את תהליך יצירת הלידים על ידי הוספת נגיעות אחרונות, נקודות נתונים לפרסונליזציה ועיצוב ייצוא לאינטגרציה חלקה עם LaGrowthMachine.

    ✅ הצלחת זרימת העבודה: זרימות העבודה שלך ב-Clay מוגדרות כעת לעבד לידים בצורה חכמה עם לוגיקה מותנית ובדיקות איכות.

5️⃣ Clay - הרצת ייצור לידים

הרץ את זרימת העבודה לייצור לידים ב-Clay ועקוב אחר התוצאות. שלב זה כולל הרצת הזרימות המוגדרות והבטחת איכות נתונים לפני ייצוא.
הרצת זרימת העבודה

הרץ את זרימות העבודה שלך ב-Clay כדי לייצר לידים מכושרים. עקוב אחר העיבוד בזמן אמת ובדוק אם יש שגיאות או בעיות איכות נתונים.
אימות תוצאות

סקור את הלידים שנוצרו לאיכות ושלמות. ודא שכל השדות הנדרשים מאוכלסים ושהנתונים עומדים בסטנדרטים של הקמפיין שלך.
הערכת איכות לידים

נתח את תוצאות ייצור הלידים, בדוק שיעורי המרה, דיוק נתונים ואיכות לידים כללית. בצע התאמות לזרימות העבודה במידת הצורך.
הכנת ייצוא

הכן את הלידים המכושרים שלך לייצוא ל-LaGrowthMachine. ודא שכל הנתונים מעוצבים כראוי ועומדים בדרישות לאוטומציית קמפיינים.

נוסחת ייצור מזהה אקראי:

javascript
Math.random().toString(36).slice(2)+Date.now().toString(36)

6️⃣ LaGrowthMachine - הגדרת קמפיין ואוטומציה

הגדר את קמפיין יצירת הקשר האוטומטי שלך ב-LaGrowthMachine באמצעות הלידים שנוצרו מ-Clay. שלב סופי זה מחבר הכל למערכת לינקדאין אוטומטית מלאה ליצירת קשר.
📧 הגדרת קמפיין
יצירת קמפיין

צור קמפיין חדש ב-LaGrowthMachine וייבא את הלידים המכושרים שלך מ-Clay. הגדר את פרמטרי הקמפיין הבסיסיים והגדרות המיקוד.
הגדרת רצף הודעות

הגדר את רצף הודעות יצירת הקשר שלך, כולל בקשות חיבור ראשוניות, הודעות מעקב ומשתני פרסונליזציה. השתמש בנתונים מ-Clay כדי לפרסן כל נקודת מגע.
הגדרות זמן ותדירות

הגדר זמון אופטימלי לפעילויות יצירת הקשר שלך. הגדר עיכובים בין הודעות, מגבלות שליחה יומיות ושיקולי אזור זמן לאפקטיביות מקסימלית.
פרסונליזציה ומשתנים

יישם פרסונליזציה דינמית באמצעות הנתונים המועשרים מ-Clay. הגדר משתנים לשמות, חברות, תפקידים ומידע ספציפי לתעשייה.
הגדרת ניטור קמפיין

הגדר מעקב וניטור לביצועי הקמפיין שלך. הגדר אנליטיקה, מעקב תגובות וניטור המרות כדי למדוד הצלחה.
סקירה סופית של הקמפיין

סקור את כל הגדרות הקמפיין, בדוק רצפי הודעות ואמת שכל האינטגרציות עובדות כראוי לפני השקת האוטומציה שלך.
השקת קמפיין

השק את קמפיין יצירת הקשר האוטומטי שלך בלינקדאין ועקוב אחר הביצועים הראשוניים. הגדר בדיקות רגילות כדי לייעל ביצועי הודעות ושיעורי תגובה.

    ✅ קמפיין פעיל: צינור האוטומציה שלך בלינקדאין כעת חי ופועל! עקוב אחר הביצועים ויעל בהתבסס על נתוני תגובה.
